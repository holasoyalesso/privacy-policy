Política de Privacidad

Política de Privacidad para la API de Buyer Persona Supercustomization 1. Introducción En la API de Buyer Persona Supercustomization, valoramos y respetamos la privacidad de nuestros usuarios. Esta Política de Privacidad tiene como objetivo explicar cómo recopilamos, utilizamos, y protegemos la información proporcionada por los usuarios de nuestra API, así como el uso de la clave API y los datos relacionados con las consultas realizadas. 2. Recopilación de Información Recopilamos y procesamos los siguientes tipos de datos: • Datos de Autenticación: La clave de API proporcionada por el usuario para autenticar y autorizar el uso de nuestra API. • Datos de Consulta: Las consultas realizadas por los usuarios a través de nuestra API, como preguntas sobre los buyer personas y sus características. • Datos de Respuesta: Las respuestas generadas basadas en las consultas proporcionadas por los usuarios. 3. Uso de la Información Utilizamos la información recopilada con los siguientes fines: • Autenticación y Seguridad: Verificar la autenticidad de las solicitudes mediante la clave API y garantizar la seguridad del acceso a los servicios de nuestra API. • Provisión de Servicios: Responder a las consultas realizadas por los usuarios con información relevante sobre los buyer personas. • Mejora de los Servicios: Analizar datos de consulta y respuesta de manera anónima para mejorar la precisión y efectividad de nuestras recomendaciones. 4. Protección de la Información Hemos implementado medidas técnicas y organizativas para proteger la información proporcionada por nuestros usuarios, incluyendo: • Encriptación de Datos: Las claves API y datos de autenticación se almacenan de manera encriptada para prevenir accesos no autorizados. • Acceso Restringido: Solo el personal autorizado puede acceder a la información sensible relacionada con las claves API y las consultas de los usuarios. • Monitoreo y Auditorías: Realizamos auditorías periódicas para identificar y solucionar posibles vulnerabilidades de seguridad. 5. Divulgación de la Información No compartimos ni vendemos la información recopilada a terceros, a menos que se requiera por ley o para proteger nuestros derechos, propiedad, o la seguridad de nuestros usuarios. 6. Derechos del Usuario Los usuarios tienen derecho a: • Acceder a la información recopilada sobre sus consultas y claves API. • Solicitar la eliminación de sus datos si ya no desean utilizar nuestros servicios. • Rectificar cualquier dato incorrecto o desactualizado relacionado con sus claves API o perfil de usuario. 7. Conservación de la Información Conservamos los datos de consulta y autenticación durante el tiempo necesario para proporcionar los servicios y cumplir con nuestras obligaciones legales y contractuales. Una vez que la información ya no sea relevante, se eliminará de nuestros sistemas de forma segura. 8. Cambios en la Política de Privacidad Nos reservamos el derecho de actualizar esta Política de Privacidad en cualquier momento. En caso de realizar cambios significativos, se notificará a los usuarios a través de los canales apropiados. 9.
